使用指南

四步上手，给你的 TRAE Work 换上新皮肤

1

安装扩展

下载 TRAE Work Skins 扩展，在 Chrome / Edge 的「扩展程序」页面中，打开右上角「开发者模式」，点击「加载已解压的扩展程序」，选择本扩展文件夹即可完成安装。

2

选择主题

点击浏览器工具栏中的扩展图标，在弹窗中选择你喜欢的主题。内置 4 款免费主题可以直接使用，更多主题可以在「主题市场」中获取。

3

从市场获取更多

点击弹窗底部的「主题市场」按钮，浏览全部主题。免费主题点击「免费获取」即可直接使用；付费主题需要购买激活码解锁。

4

解锁付费主题

在主题市场点击付费主题的「购买解锁」按钮，完成支付后获取激活码。在弹窗底部点击「输入激活码」并粘贴，验证通过后全部 3 款付费主题永久解锁。

项目简介

TRAE Work Skins 是一个为 TRAE Work 网页版（work.trae.cn）量身打造的浏览器扩展，让用户可以一键切换不同的皮肤主题，将单调的工作界面变成赏心悦目的视觉体验。

技术特点

  • 纯 CSS 注入，不修改页面原有功能
  • 图片主题采用 Blob URL 技术，规避 CSP 限制
  • 毛玻璃质感面板（backdrop-filter）
  • 激活码本地验证，无需联网

主题类型

  • 纯色渐变主题（深色 / 浅色）
  • 图片背景主题（人像 / 萌宠）
  • 动画效果（霓虹扫描、花瓣飘落等）
  • 持续更新中

开放源码

本项目基于 MIT 协议开源，欢迎开发者提交主题投稿或功能建议。如果你也设计了好看的主题，可以提交 Pull Request 加入主题市场。
