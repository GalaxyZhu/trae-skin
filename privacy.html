TRAE Work Skins 隐私政策

最后更新：2026 年 7 月 17 日

概述

TRAE Work Skins（以下简称"本扩展"）是一个浏览器扩展，用于为 TRAE Work 网页版（work.trae.cn）提供个性化皮肤主题。我们高度重视您的隐私。

数据收集

本扩展不收集、不存储、不上传任何用户个人信息。具体来说：

  • 不收集浏览历史、Cookie 或任何个人身份信息
  • 不与任何服务器通信，不发送任何遥测数据
  • 不包含任何分析、追踪或广告 SDK
  • 所有主题切换和激活码验证均在用户本地浏览器中完成

本地存储

本扩展使用 Chrome/Edge 的 chrome.storage.local API 在用户本地存储以下数据：

  • 当前选中的主题名称（如 "cyberpunk"、"sakura"）
  • 已获取的主题列表
  • 付费主题解锁状态（布尔值）

这些数据仅存储在用户本地浏览器中，不会同步到云端，不会传输到任何服务器。用户可随时通过卸载扩展清除这些数据。

权限说明

  • storage：用于在本地保存用户的主题选择偏好
  • activeTab：用于向当前打开的 work.trae.cn 页面注入主题 CSS 样式

第三方服务

本扩展不使用任何第三方服务。主题市场页面（trae-skin.galaxyzhu.workers.dev）托管在 Cloudflare Pages 上，但该页面是纯静态网站，不收集用户数据。

儿童隐私

本扩展不针对 13 岁以下儿童，也不故意收集儿童的任何信息。

变更通知

如果本隐私政策发生变更，更新后的政策将发布在本页面，并更新顶部的日期。
